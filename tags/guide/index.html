MoAI Platform Features

MoAI Platform이 제공하는 가상 AI 가속기의 동적 할당 기능으로 엔지니어는 필요한 만큼의 GPU 자원으로만 딥러닝 학습할 수 있습니다. 즉 연산이 실행되는 동안에만 GPU 자원이 할당되어 합리적인 비용으로 GPU 자원을 사용할 수 있습니다.
